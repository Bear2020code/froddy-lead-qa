# Frontend strategy

Froddy Lead QA cabinet is developed as a separate React SPA inside this repository.

We reuse Froddy public website visual language:
- colors
- spacing
- button/card style
- wording style

We do NOT couple the cabinet to the current Froddy landing implementation.

Public marketing pages may later live on the main froddy.io website.
The product cabinet may later be served from app.froddy.io or froddy.io/app/lead-qa.

V0.1 priority:
- working cabinet
- Summary / Actions / Journal
- CSV upload
- Action queue with verdict
- Decision journal
